DND Tool Set

This is my most active project, as it has actually really started to save me time. I am currently the dungeon master for 2 groups of 5 people, and preparing materials for 10 players can be difficult. While I still like to take the personal route and make things super fun for my players, having a level of automation for when things get away from me has helped. DND Tool Set isn't really one program, it's more of a collection of python scripts that I use as needed.

The tool set currently includes an encounter generator and a dungeon generator (without hallways). I have been fine tuning a bad guy maker, to make interesting villains. That will be uploaded to github shortly. I also want to make a loot generator. I am saving the loot generator for last, because it will probably be the easiest to implement.

SmartWord - CSE2010

SmartWord is a project for my Data Structures and Algorithms class. The goal of this project was to design a text prediction program using corpora. The provided corpora were tweets by then presidential candidates Donald Trump and Hilary Clinton. The program was meant to take the corpora and use it to recognize patterns, then take other tweets letter by letter and try to guess the incoming word. When the incoming word was guessed, the input skips to the next full word.

This project has been added to my curated portfolio because it helps demonstrate my earlier work with Java, my competency with programming data structures and my penchant for project organization. I am quite proud of the prediction accuracy I achieved - around 48% with the tested tweets.

SWE5415 Project

This is a currently ongoing project for my Software Testing II class. The idea is to pipe an input of matrix coordinates, and then determine if the matrix is reflexive, transitive, symmetric, one to one, and onto. In addition, if the matrix is symmetric, reflexive and transitive, it needs to be able to create partitions showing the equivalence classes. Currently only assignment 1 for the overall project is finished. Each of the above functions is separated innto its own small program. The solutions are also fairly hacky, as handling matrices is difficult with C.

I have included this in my portfolio to have a currently on-going school project available. It shows that I track requirements and documentation as I go, allowing for easy report making. I also want to show the documents that I have created for helping other students in the class.
